# Getting Started

Welcome to VMM-FRC! This section will help you get started with installing and using the software.

## Quick Start

1. Install VMM-FRC using pip or the Windows installer
2. Launch the GUI application
3. Import your CT image sequence
4. Analyze fiber orientation and volume fraction
5. Export results for further analysis
